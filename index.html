Academic Projects

AWS Amazon Recommender

Project Goal: Develop a scalable recommendation system on AWS to enhance the Amazon shopping experience.

Tech Stack Used: Python, AWS SageMaker, AWS Lambda, AWS DynamoDB, AWS S3.

Algorithms Used: Collaborative Filtering, Content-Based Filtering, Model Stacking.

Results: Successfully deployed a system with a test RMSE of 0.24, improving product recommendations for users.

Deep Learning Drowsy Driver Detection

Project Goal: Develop a deep learning model to detect driver drowsiness in real-time and enhance road safety.

Tech Stack Used: Python, TensorFlow, Keras, OpenCV.

Algorithms Used: Convolutional Neural Networks (CNN), YOLOv8, EfficientNet.

Results: Successfully achieved a test IoU of 0.75 and accuracy of 0.92, providing real-time drowsiness detection for drivers.

Spotify Music Recommender

Project Goal: Create a personalized music recommendation system to enhance user engagement on Spotify.

Tech Stack Used: Python, Spotify API, Scikit-Learn, K-Means Clustering, Graph Mining.

Algorithms Used: K-Means Clustering, Graph-Based Recommendations.

Results: Developed a recommender system with improved song and artist recommendations based on user listening habits.
