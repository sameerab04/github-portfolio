About Me

I'm a passionate data scientist with a strong foundation in machine learning and data science. Currently pursuing a Master of Science in Machine Learning and Data Science at Northwestern University, I have experience working on various projects ranging from predictive maintenance to recommendation systems. I enjoy solving complex problems using advanced data analysis techniques and deploying solutions at scale.

AWS Amazon Recommender

Developed a recommendation system using AWS services to enhance the Amazon shopping experience. It leverages content-based and collaborative filtering techniques, built and deployed on AWS infrastructure including SageMaker, Lambda, and DynamoDB.

Spotify Music Recommender

A music recommendation system built using data from Spotify. The project uses advanced data mining techniques like K-Means Clustering and Graph Mining for user segmentation and artist recommendation, aimed at improving user engagement through personalized music suggestions.

BDG2 Energy Prediction

This project focuses on predicting energy consumption in educational buildings using the Building Data Genome 2 (BDG2) dataset. The project employs machine learning models like LightGBM, Random Forest, and Neural Networks to improve energy efficiency and support sustainability initiatives.

Deep Learning Drowsy Driver Detection

A deep learning-based system designed to detect drowsiness in drivers by analyzing facial features, particularly the state of their eyes. The project uses models such as CNNs, YOLOv8, and EfficientNet, and aims to enhance road safety by providing real-time alerts to drivers.

Data Structures

This repository contains coursework related to data structures, including various assignments and lab exercises. The tasks involve creating and manipulating data structures such as linked lists, stacks, and queues, with applications ranging from vending machines to the game of Boggle.

Data Science

This repository includes assignments and projects from a course on Data Science. The work covers topics such as JSON data parsing, SQL, the Apriori algorithm, and graph databases like Neo4j. The term project focuses on an end-to-end data science problem.

Causal Learning and Inference Using LTMG for Gene Expression Levels

This project explores statistical dependencies between genes in single-cell gene expression datasets using multivariate information. It focuses on causal learning and inference to understand gene interactions better.

Computational Genomics

The repository contains work related to computational genomics, including the analysis of genetic data and the development of computational methods to study the structure and function of genomes.
